If you're looking to make an extension to ACF, please see how the [ACF missiles repository](https://github.com/ACF-Team/ACF-3-Missiles) is structured.

Glua (Gmod Lua) was developped from Lua, which has its own [official tutorial](https://www.lua.org/pil/1.html).

More information about gmod scripting can be found in the [official wiki](https://wiki.facepunch.com/gmod).
