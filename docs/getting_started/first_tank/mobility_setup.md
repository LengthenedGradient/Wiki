# Suspension tool

{: .notice}
If it helps, [Review Multi-Parent](basic_tools.html/#multi-parent).

Nothing connects the wheels to the baseplate yet.

Select the "ACF Suspension Tool".

`SHIFT + RIGHT CLICK` to select the baseplate. Any wheels you select will now target the baseplate.

`RIGHT CLICK` all your wheels in any order.

# AIO

Soon you can start driving your tank. Link your AIO controller to the gearbox.

Hold `C` then `RIGHT CLICK` the AIO controller and select "Edit AIO Settings".

Here, you can configure various settings about the AIO controller.
