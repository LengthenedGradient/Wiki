{: .notice}
Please install the [Essential ACF Builder Collection](https://steamcommunity.com/workshop/filedetails/?id=3367738921)

These addons aren't required, but we strongly recommend using them to make your building experience better.

{: .notice}
All tutorials assume the player knows how to use a physics gun and basic tools.
If you are not familliar with those, please see the "[Getting Started](https://wiki.facepunch.com/gmod/Addons)" and "[Sandbox Tools](https://wiki.facepunch.com/gmod/Tools/Balloons)" sections on the Gmod wiki.
